# Web-Development

## Purpose
This repository is used for learning and practicing full stack web development using the MERN stack through small exercises, experiments, and mini projects.

## Tech Stack (MERN)
- MongoDB
- Express.js
- React
- Node.js

## Topics Covered
- HTML, CSS, JavaScript fundamentals
- Frontend development with React
- Backend development with Node.js and Express
- Database basics with MongoDB
- Connecting frontend and backend (REST APIs)

## Status
This is a learning repository.  
Code may be incomplete, experimental, or change frequently as concepts are explored.

## How to Run
Instructions vary by folder.  
Most frontend examples can be run by installing dependencies and starting a local development server.
